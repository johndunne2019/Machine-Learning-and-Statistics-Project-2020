Welcome to Power Production output predictor



This web service allows you to enter a wind speed and have a predicted power output for that wind speed returned. The predicted power output returned is based on a Keras neural network model that I have trained using the power production data set.

Further details are available at my Github repository for this project: https://github.com/johndunne2019/Machine-Learning-and-Statistics-Project-2020
